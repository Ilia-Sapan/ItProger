

Задача 1: Корень числа
Запроси у пользователя число. Попробуй вычислить его квадратный корень. Если введено отрицательное число или некорректный ввод, обработай ошибку.

Задача 2: Конвертация строки в число
Пусть пользователь вводит строку. Попробуй преобразовать её в число (int или float). Обработай ошибку, если ввод некорректный.

Задача 3: Сложение слов
Запроси у пользователя два слова. Попробуй сложить их, как строки. Если вместо строки пользователь ввёл что-то другое, обработай это.

Задача 4: Индекс в строке
Попроси у пользователя ввести строку и число. Найди символ строки по указанному индексу. Если индекс некорректный, обработай ошибку.

Задача 5: Создание файла
Запроси у пользователя имя файла. Попробуй создать файл с этим именем. Если имя недопустимо (например, содержит запрещённые символы), обработай ошибку.

Задача 6: Деление чисел с ручным исключением
Запроси два числа. Попробуй выполнить деление, но выбрасывай исключение вручную, если результат деления меньше 1.

Задача 7: Вложенные try-except
Напиши программу, где внутри одного try-except есть другой. Например, сначала обработай ввод числа, затем попробуй выполнить деление с этим числом.

Задача 8: Работа со словарём
Создай словарь, например: {"ключ1": "значение1", "ключ2": "значение2"}. Запроси у пользователя ключ. Если ключа нет в словаре, обработай ошибку.

Задача 9: Проверка ввода числа
Запроси у пользователя число. Если ввод некорректный (например, текст), обработай ошибку и выведи сообщение, предложив повторить ввод.

Задача 10: Индексы в списке
Создай список из нескольких элементов, например: ["яблоко", "банан", "вишня"].
Пусть пользователь вводит индекс элемента списка.
Если индекс неверный (выходит за пределы списка), обработай ошибку и выведи сообщение.
